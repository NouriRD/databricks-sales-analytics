# Sales Analytics using Apache Spark & Delta Lake

## Project Description
This project demonstrates a simple end-to-end data analytics workflow
using Apache Spark on Databricks Community Edition.

## Technologies
- Databricks Community Edition
- Apache Spark (PySpark)
- Delta Lake (Managed Tables)
- Spark SQL
- Python (Matplotlib)

## Project Steps
1. Data creation and ingestion
2. Data transformation using Spark
3. Storage as a managed Delta table
4. Analytics using Spark SQL
5. Data visualization using Python

## Platform
This project was developed and tested on Databricks Community Edition.
